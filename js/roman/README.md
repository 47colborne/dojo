> Template for JavaScript dojos


## Install

```sh
$ npm install --save dojo-template
```


## Usage

```js
var dojoTemplate = require('dojo-template');

dojoTemplate('Rainbow');
```

```sh
$ npm install --global dojo-template
$ dojo-template --help
```
